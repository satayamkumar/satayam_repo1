Deadline

New Task Creation

For creating a new task click on the task creation icon in the bottom right corner.

When you click on the task creation icon, the wizard will display this screen. Here you have to enter the required details to move further.

If a pop-up with the 'Invalid Field' message appears on the screen, it means the required information is incomplete. These are the mandatory fields that need to be filled with the correct information.

Once the details are entered, Click on the "Create Task" Button Or "Create Edit Task" Button. A task will be created when you click 'Create Task,' and if you click 'Create Edit Task,' the task screen will be displayed.

When you click on Create Task

Go to the Project Section to view all the Tasks that have been created.

In the Kanban view, the project list will open, there you can easily see your created task.

Now the Task has been created

When you click on Create Edit Task

This Screen will display when you click on Create Edit Task, You can make edits in the task directly at the time of creation.
